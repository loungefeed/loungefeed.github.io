Privacy Policy

Effective date: July 1, 2026

This Privacy Policy explains how LoungeFeed Project operates LoungeFeed and handles information when users connect content sources to watch their own authorized feeds on Apple TV.

Overview

LoungeFeed is a local-first viewing product. The Apple TV app connects to a LoungeFeed Host app running on a user's Mac or iPhone. The Host handles source authorization, local pairing, local cache, and read-only content access for the paired Apple TV viewer.

LoungeFeed is designed for viewing. It does not post, like, follow, comment, save, bookmark, send messages, or perform other account-writing actions on behalf of users.

Information we access

Depending on which sources a user connects, LoungeFeed may access the following information through the source's official authorization flow or API:

  • Basic account identity, such as display name, username, and avatar.
  • Read-only feed, board, channel, pin, post, reply, message, image, and video metadata that the user is authorized to view.
  • Media URLs or media identifiers needed to display authorized images and videos inside LoungeFeed.
  • Local pairing status between the Host and the user's Apple TV viewer.

For the planned Pinterest integration, LoungeFeed intends to request only the minimum read-only scopes needed to display a user's authorized Pinterest account identity, boards, and pins.

Information we do not collect

  • LoungeFeed does not ask users to enter Pinterest passwords or other source account passwords into LoungeFeed.
  • LoungeFeed does not collect payment card numbers through this website.
  • LoungeFeed does not sell personal information.
  • LoungeFeed does not use Pinterest web scraping, Pinterest session cookies, or unofficial Pinterest browser automation for the planned Pinterest integration.
  • LoungeFeed does not create, modify, save, or delete Pinterest content.

Where data is stored

Source authorization tokens, local cache, media cache, and pairing data are stored on the user's Host device. LoungeFeed does not currently operate a cloud service that receives or stores users' source tokens, Pinterest content, or feed contents.

If LoungeFeed later adds an optional cloud service, this policy will be updated before that service is used for source account data.

How we use information

LoungeFeed uses authorized information to:

  • Show the user's selected feeds and media on Apple TV.
  • Display account connection status in the Host app.
  • Cache content locally for performance and playback reliability.
  • Protect users from unsafe or unwanted content where content safety controls apply.
  • Diagnose local Host connectivity and playback issues.

Pinterest integration

LoungeFeed's planned Pinterest integration is read-only. It will use Pinterest OAuth and official Pinterest API access after Pinterest grants the required developer access. The integration is intended to let users view their own authorized Pinterest boards and pins on Apple TV through their local LoungeFeed Host.

LoungeFeed will not use Pinterest API access to:

  • Create, edit, delete, or save pins.
  • Follow or unfollow users or boards.
  • Post comments or messages.
  • Access Pinterest passwords or session cookies.
  • Scrape Pinterest web pages.

Sharing

LoungeFeed does not sell source data or feed contents. Content is shared locally from the Host to the paired Apple TV viewer so the user can watch it. LoungeFeed may use platform services such as Apple, GitHub Pages, or domain/DNS providers to operate the app, website, or distribution infrastructure, but those services are not used to sell source account data.

User choices

  • Users can disconnect a source account inside LoungeFeed Host.
  • Users can clear local cached media and local feed data inside LoungeFeed Host where cache controls are available.
  • Users can revoke source authorization through the connected source's own account settings.
  • Users can uninstall LoungeFeed Host to remove local app data according to the operating system's app data behavior.

Children

LoungeFeed is not directed to children under 13. Users should only connect accounts they are authorized to use and view.

Changes

We may update this Privacy Policy as LoungeFeed changes. The effective date at the top of this page will be updated when material changes are made.

Contact

For privacy questions or support, contact us at loungefeed.app@gmail.com.
