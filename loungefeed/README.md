# LoungeFeed public site

This folder contains a minimal static public website for LoungeFeed and the copy deck needed for a Pinterest developer application.

The public files are:

- `index.html` - product landing page
- `privacy.html` - public Privacy Policy

The internal application files are:

- `pinterest-application.md` - copy/paste draft for Pinterest developer access
- `deployment-checklist.md` - deployment and application checklist
- `placeholders.md` - all placeholders that must be replaced before publishing or submitting

## Recommended temporary hosting

For the fastest temporary review URL, create a product-specific GitHub account and publish this folder with GitHub Pages.

Recommended setup:

- GitHub account: `https://github.com/loungefeed`
- Repository name: `loungefeed.github.io`
- Pages source: this static folder
- Temporary URL: `https://loungefeed.github.io/`

If a custom domain is available later, use the custom domain as the public URL and keep GitHub private or less visible where possible.

## Privacy posture

Do not publish personal names, personal email addresses, private addresses, or personal Git metadata.

Use a public brand/operator name such as `LoungeFeed Project` until a legal company or studio name is ready. Do not use false legal information in platform applications; only minimize what is public.

## Before publishing

Replace all bracketed placeholders listed in `placeholders.md`.
