Private diagnostic upload

Send a support report.

Upload the privacy-safe diagnostic file created by LoungeFeed Host. You will receive a short Support ID to include when contacting us.

Upload report

  1. In LoungeFeed Host, open Support Diagnostics.
  2. Choose Create diagnostic report.
  3. Select the generated .loungefeed-diagnostics.json file below.

Reports are limited to 5 MB, stored in a private Cloudflare R2 bucket, and automatically deleted after 30 days. The upload endpoint rejects credentials, cookies, request bodies, account content, media, and upstream URLs. After uploading, email the Support ID and a short description of the problem to loungefeed@outlook.com.
